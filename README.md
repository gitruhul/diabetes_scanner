# Run Locally

## Prerequisites

- Python 3+
- MySql Database

## Create Virtual Environment

```bash
  python -m venv venv
  # Activate
  .\venv\Scripts\activate.bat
```
## Install Requirements

```bash
  pip install -r .\requirments.txt
```
## Configure Properties

Open the file `config.properties` and configure all the required values.

## Run the app

```bash
  python .\app.py
```
